# Sotto MVP 0.1 Mac App 实现计划

> **面向 AI 代理的工作者：** 必需子技能：使用 superpowers:subagent-driven-development（推荐）或 superpowers:executing-plans 逐任务实现此计划。步骤使用复选框（`- [ ]`）语法来跟踪进度。

**目标：** 构建一个本地可运行的 SwiftPM + SwiftUI macOS App，完成已有稿件的切分、编辑、聚光流预览和 Mac 本机提词浮窗闭环。

**架构：** `SottoCore` 承载模型、切分、会话推进和本地保存；`Sotto` 承载 SwiftUI App、主窗口和 AppKit 提词浮窗桥接。主窗口使用 SwiftUI，提词浮窗仅在窗口层级、置顶、透明和无边框行为上使用 AppKit。

**技术栈：** SwiftPM、Swift 6、SwiftUI、AppKit、XCTest。

---

## 文件结构

- 创建 `Package.swift`：定义 `SottoCore` library、`Sotto` executable 和 `SottoTests`。
- 创建 `Sources/SottoCore/Models`：稿件、句子、短语、速度、会话、浮窗设置模型。
- 创建 `Sources/SottoCore/Services`：确定性切分、手动切分、本地 JSON 保存。
- 创建 `Sources/SottoCore/Stores`：最近稿件 Store。
- 创建 `Sources/Sotto`：SwiftUI App、首页、编辑页、预览、切分面板、浮窗桥接。
- 创建 `Tests/SottoTests`：覆盖分句、短语、手动切分、计时、会话推进、本地保存。
- 创建 `script/build_and_run.sh` 与 `.codex/environments/environment.toml`：提供 Codex Run 和本机验证入口。
